# WalaSpringBoot2020
Proyecto de como hacer una tienda de segunda mano usando Spring Boot y otras cosillas

## Tecnologías
* H2 como base de datos para que quien lo quiera no usar MySQL, luego lo cambiaré
* Spring Security
* Spring Sessions
* Tyemeleaf
